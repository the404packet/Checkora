# Issue Assignment Policy

## Assignment Window

Contributors have 36 hours from assignment to open a pull request.

## Reminder System

A reminder comment is automatically posted after 24 hours if no PR has been opened.

## Automatic Reassignment

If no pull request is opened within 36 hours:

- Assignee is removed
- Issue becomes available for reassignment
- A notification comment is posted

## Extensions

Contributors may request additional time by posting a progress update comment on the issue.
